第8章 基本登录功能

发表于 2018-01-25 | 分类于 railstutorial

登录表单

session 会话不是模型, 因此不能创建类似 @user 的变量.
对会话来说, 我们需要指明资源的名称以及相应的 URL.

flash.now

flash.now 专门用于在重新渲染的页面中显示闪现消息; 与 falsh 不同的是, flash.now 中的内容会在下
次请求时消失.

include

在控制器的基类(ApplicationController)中引入辅助方法模块 SessionsHelper, 其中的定义的方法可
以在控制器中使用.

log_in 方法

session[:user_id] = user.id 会在用户的浏览器中创建一个临时 cookie, 内容是加密后的用户 ID.
session 方法创建的是临时会话, 浏览器关闭后立即失效.

current_user 方法

||= 运算符:
@current_user ||= User.find_by(id: session[:user_id])
几乎等效于 @current_user = @current_user || User.find_by(id: session[:user_id])
只有当 @current_user 为假时才会执行后面的表达式; 如果 @current_user 为真, || 会终止执行, 这种方式叫短路计算.

获取固件用户

user = users(:michael) 在测试中获取这个用户

is_logged_in?

is_logged_in? 辅助方法和 logged_in? 功能一样, 测试中不能使用 current_user 方法, 但是可以使用 session 方法. (定义不同的名字以免混淆)
